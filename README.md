# DB設計君 ver.1.2

Laravel、MySQLに対応したDB設計用テンプレートマクロ。
Google Apps Scriptマクロを使用したmigrationコード生成機能により、そのまま貼り付けて使用可能。
各要素を修正の際も、自動でコード修正を行い、ミスを軽減。
